# Task 05 - Problem

## Description

Read two integers **R** and **C**, then read an **R × C** table of integers into `int[][]`.

Print:

- Sum of each row (one per line)
- Sum of each column (one per line)
- Total sum of all values

---

## Input

- Two integers **R** and **C**
- Followed by **R × C integers**

---

## Output

- Row sums (one per line)
- Column sums (one per line)
- Total sum

---

## Constraints

- \( 1 \leq R, C \leq 50 \)
