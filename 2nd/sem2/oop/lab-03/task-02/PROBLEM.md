# Task 02 - Problem

## Description

Read an integer **N** and then **N integers**.  
Read an integer **k**.

Produce a new array that is the original array **shifted right by k positions** (with wrap-around).

---

## Example

Input array:
[10, 20, 30, 40, 50]

k = 2

Output:
[40, 50, 10, 20, 30]

---

## Input

- An integer **N**
- Followed by **N integers**
- Then an integer **k**

---

## Output

- The shifted array printed on one line
