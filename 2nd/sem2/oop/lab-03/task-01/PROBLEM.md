# Task 01 - Problem

## Description

Read an integer **N**, then read **N integers** into an array.  
Compute and print:

- **Minimum**
- **Maximum**
- **Sum**
- **Average** (double, 2 decimal places)
- **Count of even numbers**

---

## Input

- An integer **N**
- Followed by **N integers**

---

## Output

Print the following values (clearly labeled):

- Min
- Max
- Sum
- Average
- Even count

---

## Constraints

- \( 1 \leq N \leq 10{,}000 \)
