# Task 03 - Problem

## Description

Write a function that:

- Returns how many times **target** appears in array **a**.

In `main`:

- Read an integer **N**
- Read **N integers** into an array
- Read an integer **target**
- Call the function
- Print the result

---

## Input

- An integer **N**
- Followed by **N integers**
- An integer **target**

---

## Output

Print:

target appears X times
