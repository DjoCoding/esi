# Task 7: Input Validation Loop (do-while)

**Description:**  
Repeatedly prompt the user for a number in the range `[10, 50]`. Keep asking until a valid number is entered. Then print the square of the valid number.

**Input:**  
Integers from the user until valid.

**Output:**  
The accepted number and its square.

**Constraints:**  
Must use a `do-while` loop.
