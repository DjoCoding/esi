# Task 3: Menu-Based Unit Converter (switch-case)

## Description

Build a console menu that converts a value based on user choice:

1. Celsius → Fahrenheit
2. Fahrenheit → Celsius
3. Kilometers → Miles
4. Miles → Kilometers

Read the option (1–4) and a numeric value, then output the converted result.

## Input

- `option` (int)
- `value` (double)

## Output

Converted value (2 decimals) and a short label.

## Constraints

If the option is not 1–4, print `Invalid option`.

---

## Example Usage

```bash
java Main 1 25
```
