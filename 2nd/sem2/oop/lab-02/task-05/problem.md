## Task 5: Check Balanced Parentheses

**Description:**  
Read a string and check if the parentheses `(` and `)` are balanced.

**Examples:**

- `(a+(b))` → `BALANCED`
- `())(` → `NOT BALANCED`

**Input:**

- One line (a single string)

**Output:**

- `BALANCED` or `NOT BALANCED`

**Constraints:**

- Only parentheses matter
- All other characters are ignored
