## Task 1: String Compression

**Description:**  
Read a string and compress consecutive characters into `char + count`.

**Examples:**

- `aaabbc` → `a3b2c1`
- `AAAAaa` → `A4a2`

**Input:**

- One line (a single string)

**Output:**

- Encoded string

**Constraints:**

- Preserve case (uppercase and lowercase are different)
- Preserve symbols (do not ignore special characters)
