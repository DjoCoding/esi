## Task 4: Print Words Line by Line

**Description:**  
Read a line and output each word on a new line, ignoring extra spaces.

**Example:**  
Input: `" I love Java "`

Output:
`I`
`love`
`Java`

**Input:**

- One line (a single string)

**Output:**

- One word per line
- Total word count

**Constraints:**

- Do not use `split()`
